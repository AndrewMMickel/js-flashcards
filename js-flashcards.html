JavaScript

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

Operators in JavaScript are very similar to operators that appear in other programming languages. The definition of an operator is a symbol that is used to perform an operation.

Variables

Variable means anything that can vary. JavaScript includes variables which hold the data value and it can be changed anytime.

Functions

a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it.

Methods

actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Arguments

In JavaScript functions are the primary modular units of execution. Arguments are values passed to the function when it is invoked.
